✨ I am a Full-Stack Developer with an emphasis on Back-End and solid experience in developing REST APIs using Java and Spring Boot, as well as skills in Swagger for API documentation and integration testing with JUnit. I have knowledge of SQL and NoSQL databases, employing Object-Relational Mapping (ORM) as well as direct JDBC access, ensuring flexibility and performance in various scenarios.
